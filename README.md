# node-gif

> Show gifs in the terminal

Works in any terminal that supports colors.

<img src="https://raw.githubusercontent.com/pau1fitz/node-gif/master/example.gif">

## Install

```
$ npm install -g node-gif
```

## Usage

```js
$ node-gif [gifSearch]
```

## Example

```js
$ node-gif funny
```
